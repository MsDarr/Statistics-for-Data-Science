# Statistical Inference

This project demonstrates the application of statistical methods using Python, focusing on probability distributions, hypothesis testing, and goodness-of-fit analysis.

## Files
- Statisticalnf.ipynb → Jupyter Notebook with full analysis
- Statisticalnf.pdf → Final report with visualizations
- notes.txt → Supporting calculations and explanations

## Key Topics Covered

### 1. Portfolio Risk Analysis (CAPM)
- Modeled portfolio returns using normal distribution
- Probability of loss ≈ 32.8%
- Top 15% return threshold ≈ 48.9%

### 2. Hypothesis Testing
- Z-test to evaluate HR reward system effectiveness
- Z-statistic ≈ -4.47
- Result: Reject null hypothesis

### 3. Type II Error Analysis
- Probability of missing true improvement ≈ 4.46%

### 4. Geometric Distribution
- Modeled expected probabilities

### 5. Chi-Square Goodness-of-Fit Test
- Chi-square ≈ 4.61
- p-value ≈ 0.5946
- Conclusion: Data fits the geometric distribution
