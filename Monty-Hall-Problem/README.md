# Monty Hall Problem

This project explores the Monty Hall problem, a classic probability puzzle, using both theoretical analysis and simulation techniques in Python.

## Files
- MontyHall.ipynb → Jupyter Notebook with simulation and analysis
- MontyHall.pdf → Final report with explanation and results

## Problem Overview

The Monty Hall problem is a probability puzzle based on a game show scenario:

- There are three doors: one hides a car, and the other two hide goats.
- The player selects one door.
- The host, who knows what is behind the doors, opens another door revealing a goat.
- The player is given the option to switch or stay.

## Key Insights

### 1. Probability Theory
- Initial probability of selecting the correct door: **1/3**
- Probability of winning after switching: **2/3**

### 2. Simulation Analysis
- Multiple simulations confirm that switching doors significantly increases the probability of winning.

### 3. Decision Strategy
- Best strategy: **Always switch**
